# PaperInterpretation
每日前沿论文解读仓库，自动同步每日精选的AI/ML领域高质量论文解读，由AI自动生成。

## 📌 项目说明
- 每天从 [alphaXiv](https://www.alphaxiv.org/) 精选1篇最新发布的高质量前沿论文
- 解读内容包含：背景介绍、核心技术方案、实验结果、工程启示、局限性分析等完整结构
- 自动同步到飞书文档 + GitHub仓库双备份

## 📅 更新规则
- 工作日每日上午9:30开始执行更新，10:00前完成
- 节假日会根据实际情况调整更新频率

## 📁 文件命名规则
`YYYY-MM-DD_论文标题.md` 例如：`2026-03-11_Agentic_Critical_Training.md`
